A simple RESTful chat backend in PHP using Slim 4 and SQLite.
Supports public chat groups where users can join groups, send messages, and view messages. Users are identified by a simple username (no registration/login system).

FEATURES:
1. Create users
2. Create public chat groups
3. Join groups
4. Send messages in groups
5. List messages in a group
6. JSON REST API
7. SQLite database (chat.db) for persistence
